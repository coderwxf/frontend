只要对象 B 满足 对象 A 的结构特征，TypeScript 就认为对象 B 兼容对象 A 的类型，这称为“结构类型”原则（structural typing）

即 B可以有A中没有的特征，但A中有的特征，B也必须要存在，此时B是子类型，A是父类型

TS中赋值规则使用的是协变，凡是可以使用父类型的地方，都可以使用子类型，即子类型兼容父类型

```ts
type A = {
  x: number;
};

type B = {
  x: number;
  y: number;
};

const B = {
  x: 1,
  y: 1
};

const A:{ x: number } = B; // 正确
```
