解构的作用是直接从对象中提取属性

```ts
const {id, name, price} = product;

// 本质是
const id = product.id;
const name = product.name;
const price = product.price;
```

```ts
let { x: foo, y: bar } = obj;

// 等同于
let foo = obj.x;
let bar = obj.y;
```



`在ts中使用`

```ts
let { x: foo, y: bar }: { x: string; y: number } = obj;
```
