## option api

Vue2中vue是一个类，创建应用就是创建Vue的实例

可以指定多个vue应用 => 创建多个vue实例即可

```js
const vm = new Vue()
```

在`new Vue`时需要传入配置对象



每一个配置项就是一个个option => vue2的API也被叫做option api 「 配置式api 」





## 挂载点

渲染后的内容需要被挂载到页面上某一个元素中，这个元素就是挂载点

+ `vue` 一般是`div#app`
+ `react` 一般是 `div#root`

数据驱动框架渲染后的内容会替换原本挂载点中的内容

使用数据驱动框架开发的多个页面网站被称之为应用 

+ 挂载点中的内容才被称之为应用
+ 挂载点本身并不是应用的一部分
