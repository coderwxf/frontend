## 什么是路由

1. 根据URL找到对应组件
2. 将匹配到的组件渲染到特定位置上「路由出口」

`router.js`

```js
import { createRouter, createWebHistory } from 'vue-router'

// 创建路由
const routes = [
  {
    path: '/cpn-a',
    component: () => import('./components/CpnA.vue')
  },
  {
    path: '/cpn-b',
    component: () => import('./components/CpnB.vue')
  },
]


// 创建路由表
const router = createRouter({
  history: createWebHistory(), // 路由模式
  routes
})

export default router
```

`挂载路由`

```js
import { createApp } from 'vue'
import App from './App.vue'
import router from './router.js'

const app = createApp(App)

// 通过use挂载路由
app.use(router)

app.mount('#app')
```

`设置导航`

```vue
<template>
  <div>
    <!--
      路由导航
      1. router-link 会被渲染为 a链接
      2. 实现页面无刷新跳转以及URL的生成和编码
    -->
    <router-link to="/cpn-a">CpnA</router-link>
    <router-link to="/cpn-b">CpnB</router-link>

    <!-- 路由出口 -- 路由占位 -->
    <router-view></router-view>
  </div>
</template>
```

`组件内获取`

```vue
<script setup>
  import { useRoute, useRouter } from "vue-router";

  const route = useRoute(); // 当前路由实例 -- this.$route
  const router = useRouter(); // 当前路由表实例 -- this.$router

  console.log(route)
  console.log(router)
</script>
```
