作用 => 找到对应元素 并 设置样式

CSS 类名没有绝对的命名规则 「 推荐使用`kebab-case` 」



| 选择器       | 例子           |
| ------------ | -------------- |
| 通用选择器   | `*`            |
| 元素选择器   | `div`          |
| 类选择器     | `.foo`         |
| ID选择器     | `#uid`         |
| 属性选择器   | `[disabled]`   |
| 后代选择器   | `div span`     |
| 兄弟选择器   | `.lorem + div` |
| 交集选择器   | `div.lorem`    |

## 通用选择器

1. 为所有元素设置通用样式 「 清除body默认的8px外边距 」

通用选择器应用通用样式于所有元素 「 某些通用样式对某些元素是不生效或者没有必要的 」

推荐单独为元素分别进行设置或使用第三方样式重置库 「 例如`normalize.cs` 」

```css
div,
p,
ul,
li {
  margin: 0;
  padding: 0;
}
```



## 属性选择器

```css
[title] /* 属性名为title */

[title='Lorem ipsum'] /* 属性名为title && 属性值为lorem */
[title=Lorem] /* 单个单词 => 可以省略引号 */

[class*=box] /* key为class value中包含box */
[class^=box] /* value以box开头 */
[class$=box] /* value以box结尾 */

[class|=dv] /* value存在 dv 或 dv-* */
[class~=dv] /* value存在 dv 「 不匹配 dv-* 」 */
```



## 后代选择器

后代元素(子代元素) = 直接后代元素 + 间接后代元素

后代元素 => `div span`

直接后代元素 => `.foo > .bar`



## 兄弟选择器

1. 默认选中的都是后继兄弟 「 选中前置兄弟 需要借助`:not` 」
2. 相邻兄弟「 直接后继兄弟 」 => `.lorem + div`
3. 全（后继）兄弟 => `.lorem ~ div`



## 并集选择器

```css
div, /* 推荐每个条件单独使用一行表示 */
.lorem {
  
}
```



## 伪类选择器

1. 用于选择处于特定状态的元素

2. 使用`:`开头

| 常见伪类     | 举例                                                         |
| ------------ | ------------------------------------------------------------ |
| 动态伪类     | `:link`、`:visited`、`:hover`、`:active`、`:focus`           |
| 目标伪类     | `:target` 「 元素ID 和 URL的fragment 进行匹配 」             |
| 语言伪类     | `:lang( )` 「 和lang属性匹配 」                              |
| 元素状态伪类 | `:enabled`、`:disabled`、`:checked`、`:selected`             |
| 结构伪类     | `:nth-child(n)`、`:nth-last-child(n)`、<br />`:nth-of-type(n)`、`:nth-last-of-type(n)` <br />`:first-child`、`:last-child`、<br />`:first-of-type`、`:last-of-type`<br /> `:root`、`:only-child`、`:only-of-type`、`:empty` |
| 否定伪类     | `:not(selector)` 「 一般需要前缀，以减小影响范围 」          |
| 其它伪类     | `:placeholder`、`:is(selector)`、`:where(selector)`、`:has(selector)` |



### 动态伪类

1. 直接给`a`设置样式，这些样式会应用于所有状态（`a:link`、`a:visited`、`a:hover`、`a:active`、`a:focus`）

2. `link`和`visited`只能用于`a`元素，其余伪类可以用于任意元素上

3. 对于同一个元素，可能同时存在多种动态伪类 「 注意设置样式的先后顺序 」

   例如a元素处于`hover`状态时

   + 同时存在`visited`或`link`中的某一种
   + 可能同时存在`focus`状态 或 `active`状态 「 按下时即满足`hover`又满足`active` 」

4. 通常推荐的顺序是：`link` -> `visited` -> `hover` -> `focus` -> `active`



### 语言伪类

`lang`用于指定元素内容使用那种语言 

`lang`是全局属性 => 一般设置在`html`元素上 => 以便全局生效

```css
/* 
	1. 参数是语言代码对应的字符串
	2. zh => 匹配所有以zh开头的语言前缀 => 例如 zh、zh-CN、zh-TW 等
*/
:lang(zh)
```



### 结构伪类

#### 基本使用

```css
:nth-child(n) /* n是从0开始递增，但是HTML元素对应索引是从1开始递增 */

/* 偶数列 */
:nth-child(2n) <=> :nth-child(even)

/* 奇数列 */
:nth-child(2n + 1) <=> :nth-child(odd) <=> :nth-child(2n - 1) 

:nth-child(1) <=> :first-child

:nth-child(-n + 3) /* 前三个元素 */

:nth-child(4n + 1) /* 设置第一个，第五个，第九个，第十三个 ... 元素 */

:root <=> html
```



#### nth-child vs nth-of-type

```html
<style>
.lorem :nth-child(2) {
  color: red;
}
</style>

<div class="lorem">
  <div>content1</div>
  <div>content2</div> <!-- 变红 -->
  <div>content3</div>
  <p>content1</p>
  <p>content2</p>
  <p>content3</p>
</div>
```

```html
<style>
.lorem :nth-of-type(2) {
  color: red;
}
</style>

<div class="lorem">
  <div>content1</div>
  <div>content2</div> <!-- 变红 -->
  <div>content3</div>
  <p>content1</p>
  <p>content2</p> <!-- 变红 -->
  <p>content3</p>
</div>
```



#### is vs where

```css
:is(.class1, .class2, .class3) > p {
  color: red;
}

/* 等价于 */

.class1 > p,
.class2 > p,
.class3 > p {
  color: red;
}
```

```css
:where(.class1, .class2, .class3) > p {
  color: red;
}

/* 等价于 */

.class1 > p,
.class2 > p,
.class3 > p {
  color: red;
}
```

`:is(selector)`和`:where(selector)`的区别是

1. `:is`的优先级「特异性」 由`selector`中最高的那个决定
2. `:where`的优先级 始终为 0

```css
/* `:is(#id, .class, article)` 的特异性是 0100，因为 `#id` 的特异性最高 */
:is(#id, .class, article) {
  margin: 20px;
}
```

```css
/* article 的特异性是 0001 */
article {
  margin: 10px;
}

/* :where(article, section, aside) 的特异性是0000 */
:where(article, section, aside) {
  margin: 20px;
}

/* 最终article元素的margin值为10px */
```



#### has

```html
<style>
 /* :has选择器较新，需要注意浏览器兼容性问题 */
.container:has(span) {
  background-color: yellow;
}
</style>

<!-- .container背景会变黄 -->
<div class="container">
  <p>Paragraph 1</p>
  <span>Span 1</span>
</div>

<!-- .container背景不会变黄 -->
<div class="container">
  <p>Paragraph 2</p>
</div>
```

```css
/* 选择包含 <p>、<span> 或 <h3> 元素中的任意一个的 .box 元素 */
.box:has(p, span, h3) { 
  display: none;
}
```

```css
/* 找满足div + .lorem的div元素 => div元素后边直接兄弟节点有样式.lorem */
div:has(+ .lorem) {
  color: red;
}
```



## 伪元素

1. 伪元素可以使用`:`开头，也可以使用`::`开头 => 推荐使用`::`开头 => 区分伪元素
2. 选择元素的某一部分
3. 伪元素对行内替换元素无效 => 设置在了占位元素上，会被实际渲染元素替换掉

常见伪元素

+ `::first-line`
+ `::first-letter`
+ `::before`
+ `::after`
+ `::-webkit-scrollbar`



`::before`和`::after`用来在一个元素的内容之前或之后插入`其他内容`

+ 插入的是装饰性内容 「 无法被选中 」
+ 即使插入的不是内容，`content`属性也不能省略，可以设置其值为空字符串
+ 默认插入的元素都是`行内非替换元素`



```css
/* -webkit前缀 => 非标属性 => 仅在webkit内容和blink内容浏览器中生效 */
/* -webkit-scrollbar及其子系列属性 => 设置滚动条样式 => 一般用于清除默认的滚动条样式  */
.lyrics::-webkit-scrollbar {
  display: none;
}
```
