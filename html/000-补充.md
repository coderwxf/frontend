## link

1. 在`head`中使用
2. 作用 => 引入外部资源文件

```html
<!-- 
	引入外部样式表 
	+ ref - relationship - 外部资源类型
	+ href - hypertext reference - 资源路径
-->
<link rel="stylesheet" href="./style.css">

<!-- 
	引入站点图标「 favicon.ico 」
	原则上站点图标可以是任意格式图片 「 推荐优先使用ICO格式 」
-->
<link rel="icon" href="./favicon.ico">
```

```html
<!-- 
	预解析（或预加载）
	1. 在当前页面资源加载完毕后
	2. 浏览器会主动提前加载并缓存将来可能需要的资源
	3. 而不是等到实际使用时再进行加载
-->

<!-- dns-prefetch 预解析域名 -->
<link rel="dns-prefetch" href="http://www.google.com">

<!-- prefetch 预加载资源 -->
<link rel="prefetch" href="./images/avatar.jpg">
```
