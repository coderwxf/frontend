react中，子传父是通过函数类型props来实现的



`父组件`

```jsx
import React, { Component } from 'react'
import Child from './Child'

export class App extends Component {
  state = {
    count: 0
  }

  render() {
    return (
      <>
        <div>{ this.state.count }</div>
        {/*
          这里的onClick事件是自己定义的, 是由我们自己触发的
          + 1. this指向是App组件
          + 2. 参数是我们自己传递的 「 我们需要自己手动传递事件参数 」
        */}
        <Child onClick={count => this.setState({ count })} />
      </>
    )
  }
}

export default App
```



`子组件`

```jsx
import React, { Component } from 'react'
import PropTypes from 'prop-types'

export class Child extends Component {
  static propTypes = {
    onClick: PropTypes.func // onClick需要是一个函数
  }

  static defaultProps = {
    onClick: () => {} // 设置默认值
  }

  render() {
    return (
      <button onClick={() => this.props.onClick(20)}>click me</button>
    )
  }
}

export default Child
```
