在JavaScript，有且仅有四种检测数据类型的方法

| 方法                             | 说明                               |
| -------------------------------- | ---------------------------------- |
| [typeof A](001-typeof%20A.md)    | 检测数据类型的`运算符`             |
| A instanceof B                   | 基于当前实例是否属于某个类进行检测 |
| constructor                      | 基于构造函数进行检测               |
| Object.prototype.toString.call() | 检测数据类型 `最好` 的方式         |
