## while

一般来说，当我们不确定循环次数时，会使用while循环

而当我们可以大致确定循环次数时，通常会使用for循环

```javascript
while (condition) {
    // 代码块
}
```

每次循环时，首先判断条件是否成立，如果成立，就执行代码块。执行完代码块后，再次判断条件是否成立，如此循环下去，直到条件不再成立。

需要注意的是，如果条件永远为真，循环就会一直执行下去，形成死循环。例如：

```javascript
while (true) {
    // 代码块
}
```



## 循环控制的关键字

1. **break**：直接结束整个循环。
2. **continue**：结束本次循环，继续下一次循环。
